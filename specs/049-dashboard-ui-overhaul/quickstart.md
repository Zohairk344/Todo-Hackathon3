# Quickstart: Dashboard UI Overhaul

## 1. Install Dependencies
Run the following command in the `frontend` directory:
```bash
npm install framer-motion
```

## 2. Verify Global Styles
Ensure `frontend/app/globals.css` is loaded and Tailwind 4 is active.

## 3. Run Development Server
```bash
npm run dev
```

## 4. Verification Steps
1. Navigate to `/dashboard`.
2. Observe the radial dark background.
3. Add a task and verify the staggered entry animation.
4. Hover over a task card to see the smooth scale and glow transition.
5. Check the header to ensure "Add Category" button is removed.
